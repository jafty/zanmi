Politique de Confidentialité

Dernière mise à jour : 11 janvier 2025

1. Introduction

Chez Rafiki Events, nous attachons une grande importance à la confidentialité et à la sécurité de vos données personnelles. Cette Politique de Confidentialité explique comment nous collectons, utilisons, et protégeons vos informations personnelles lorsque vous utilisez notre plateforme.

2. Données que nous collectons

Nous collectons différentes catégories de données personnelles :

  • Données fournies par l'utilisateur : Nom d'utilisateur, adresse e-mail, avatar, description de profil, etc.
  • Données liées aux événements : Participation, messages aux organisateurs, localisation (si fournie).
  • Données techniques : Adresse IP, navigateur utilisé, logs de connexion.

3. Pourquoi collectons-nous vos données ?

Vos données sont collectées dans les buts suivants :

  • Permettre la création et la gestion d'événements.
  • Faciliter les paiements sécurisés via Stripe.
  • Envoyer des notifications (participation acceptée, nouvelles demandes, etc.).
  • Améliorer nos services et votre expérience utilisateur.

4. Partage de vos données

Nous ne vendons jamais vos données à des tiers. Cependant, certaines données peuvent être partagées avec :

  • Stripe : Pour traiter les paiements sécurisés.
  • Autorités légales : Si requis par la loi.

5. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Accès à vos données personnelles.
  • Correction de vos informations en cas d'erreur.
  • Suppression de vos données ("droit à l'oubli").
  • Opposition au traitement de certaines données.
  • Portabilité de vos données.

Pour exercer ces droits, contactez-nous à l'adresse suivante : contact@rafiky.fr.

6. Sécurité de vos données

Nous utilisons des mesures techniques et organisationnelles pour protéger vos données, notamment :

  • Chiffrement des mots de passe (bcrypt).
  • Connexion via HTTPS pour sécuriser les échanges.
  • Accès restreint à la base de données.

7. Conservation des données

Vos données personnelles sont conservées uniquement le temps nécessaire à la réalisation des finalités décrites dans cette politique. Les comptes inactifs depuis plus de 2 ans seront automatiquement supprimés, sauf obligation légale contraire.

8. Cookies

Notre site utilise des cookies pour :

  • Gérer votre session de connexion.
  • Analyser le trafic et améliorer nos services (via Google Analytics).
Vous pouvez gérer vos préférences de cookies via les paramètres de votre navigateur.

9. Contact

Si vous avez des questions concernant cette Politique de Confidentialité, vous pouvez nous contacter à :

  • Email : contact@rafiky.fr
  • Adresse : 414 route de Saint-Simon 31100 Toulouse, France
